了解 ECMAScript 标准中的数据类型

一篇实验册要素：

瞎掰：

计算机中的数据类型描述的是术语--数据结构和算法中的数据，目的是给数据进行分类，对不同类型的数据进行不同的处理。

那么问题来了，为什么要对数据进行分类呢？

嗯，分类思想在生活中还是能找到一些例子：人分男女，画分素描速写与油画，翻看各类电商网站不难发现有书籍食品等分类。

去化学中找一些例子：往大了分可以有有机物，无机物的分法；往小了说，醇，酚，醚等的分类方案也为人所知晓。

分类是为了能对一个特定的群体抽象出共有的特点，通过对群体中的某个个体进行研究可以推论出整个群体在某些情景下的特点。

实验目的：

了解 ECMAScript 标准中的数据类型，了解数据社会中的“数据关系”。

实验日期：

实验原理：

20200212 消息，最新的 ECMAScript 标准定义的数据类型有8种，我朋友说他想看看。

按类似男女的分类，数据被分成了两种类型： 原始类型[primitive values]和引用类型。他们的区别，下个实验见，浏览器见。

另一种就是比较细的分类： Boolean, Null, Undefined, Number, BigInt, String, Symbol, Object。
