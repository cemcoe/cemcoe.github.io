

# Build a Tribute Page
https://www.freecodecamp.cn/challenges/build-a-tribute-page

# 要求

- 每个诗人心中都有自己的致橡树，by 舒婷。

- 目标: 设计一个类似于: https://codepen.io/FreeCodeCamp/full/NNvBQW/的页面。
